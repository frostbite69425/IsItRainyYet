# Is It Rainy Yet - A Weather App

A weather application built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## About

This project fetches real-time weather data from a public API and displays it to the user. The primary learning objective is working with **asynchronous JavaScript** — specifically Promises and the `async/await` syntax.

## Concepts Practiced

- Promises and the Promise chain (`.then()`, `.catch()`, `.finally()`)
- `async/await` for cleaner async flow
- `fetch` API for HTTP requests
- Handling loading and error states
- Parsing and displaying JSON data

## Features

- Search weather by city name
- Displays temperature, conditions, humidity, and wind speed
- Toggle between Celsius and Fahrenheit
- Error handling for invalid city names or failed requests
